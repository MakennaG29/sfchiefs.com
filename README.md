# sfchiefs.com

Welcome to the repository for **sfchiefs.com**! This website is built using HTML5 and hosted via GitHub Pages using a custom domain.

🔗 **Live Website**: [https://sfchiefs.com](https://sfchiefs.com)

## 🎮 Features
* Smooth HTML5 game performance
* Clean, ad-free interface
* Accessible on school and work networks
* Custom domain integration

## 🛠️ How to Add Games
1. Download an HTML5 game folder containing an `index.html` file.
2. Upload the folder to this repository.
3. Link the game folder in the main navigation menu.
