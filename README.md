## 🎮 SF Chiefs Games Hub

A collection of fun, browser-based games that work at school! All games run directly in your browser with no external dependencies.

### 🎯 Available Games

1. **2048** - Combine numbered tiles to reach 2048
   - Use arrow keys to move tiles
   - Track your best score
   - Classic puzzle game

2. **Snake** - Classic snake game
   - Use arrow keys to control the snake
   - Collect food to grow
   - Avoid hitting walls and yourself

3. **Tic Tac Toe** - Play against the computer
   - Click cells to make your move
   - Beat the AI or draw
   - Quick fun game

4. **Memory Match** - Test your memory
   - Flip cards to find matching emojis
   - Match all pairs to win
   - Great for concentration

5. **Trivia Quiz** - Answer trivia questions
   - 10 random questions each game
   - See your score at the end
   - Learn new facts

6. **Hangman** - Guess the word
   - Click letters to guess
   - Guess the word before you lose
   - Classic word game

### 🚀 How to Use

1. Open `index.html` in your web browser
2. Click on a game from the menu
3. Click the game buttons in the top navigation to switch games
4. Press "New Game" to restart
